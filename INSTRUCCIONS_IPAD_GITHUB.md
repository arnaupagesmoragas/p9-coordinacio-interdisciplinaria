# Instruccions ràpides per iPad i GitHub Pages

1. Descomprimeix el ZIP a l'app Fitxers.
2. Crea un repositori públic nou a GitHub.
3. Puja aquests fitxers descomprimits, no el ZIP:
   - index.html
   - README.md
   - INSTRUCCIONS_IPAD_GITHUB.md
   - manifest.webmanifest
   - .nojekyll
4. Ves a Settings > Pages.
5. Selecciona Deploy from a branch.
6. Selecciona main i /root.
7. Desa i espera uns minuts.
