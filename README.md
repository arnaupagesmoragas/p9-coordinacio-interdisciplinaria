# Llar amb suport Sant Andreu

Sistema intern de seguiment i coordinació en format web estàtic.

## Estructura

- `index.html`: portada i accés a les eines.
- `diari.html`: diari de seguiment.
- `incidencies.html`: registre d’incidències.
- `reunions.html`: reunions de coordinació.
- `mediacio.html`: mediació.
- `DADES_APP.md`: font de contingut.
