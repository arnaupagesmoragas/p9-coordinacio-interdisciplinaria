# Llar amb suport Sant Andreu · Sistema intern de seguiment

Aplicació web estàtica per al seguiment intern de la Llar amb suport Sant Andreu.

Inclou:
- Fitxa del recurs i context residencial.
- Perfils operatius de les residents.
- Registres de diari de seguiment.
- Registre d'incidències.
- Reunions de coordinació.
- Cas de convivència Sara-Laura.
- Sessió de mediació, acords i traçabilitat.
- Filtres, consulta global, edició, duplicació, eliminació, exportació CSV/JSON i importació JSON.

Ús local:
1. Obrir `index.html` amb Safari, Chrome, Edge o Firefox.
2. Les dades es desen al navegador del dispositiu.
3. Per publicar a GitHub Pages, pujar aquests fitxers descomprimits a l'arrel del repositori.
